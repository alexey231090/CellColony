# Технический справочник CellColony (Reference)

## 🎮 Описание игры
**CellColony** — это минималистичная стратегия в реальном времени (RTS), вдохновленная Spore и Auralux.
Игрок управляет колонией клеток, захватывая мир через передачу энергии.

## 🏗 Ключевые системы

### 1. Клетка (BaseCell) — CharacterBody2D
- **Логика:** Физический объект. Поддерживает столкновения и расталкивание (Separation).
- **Механики:** 
  - `command_attack(pos, target)`: Атака + движение к цели.
  - `command_move(pos)`: Только движение, прекращение огня.
- **Физика:** Использует `move_and_slide()` для плавного обтекания препятствий.

### 2. Модуль передвижения (MoverModule)
- **Логика:** Рассчитывает `velocity` для родительской клетки.
- **Особенности:** Включает систему расталкивания (Separation), которая не дает клеткам слипаться и позволяет им толкать нейтральные объекты. Плавный разгон и инерция (водная среда).

### 3. Модуль стрельбы (ShooterModule)
- **Логика:** Автонаведение на движущуюся цель (`target_node`) или ближайшего врага в радиусе.
- **Автоатака:** Сканирует область в 600px; если ручной цели нет, выбирает ближайшего врага/нейтрала.
- **Умное прекращение огня:** Останавливается, если цель — союзник с полной энергией. Нейтральные клетки не стреляют.

### 4. Камера (CameraFollow)
- **Логика:** Центрируется на "центре масс" всех клеток игрока.
- **Динамический зум:** Автоматически отдаляется, чтобы вся колония игрока была в кадре.

### 5. Менеджер выбора (SelectionManager)
- **Клик по клетке:** Устанавливает глобальную цель для атаки/лечения.
- **Клик в пустоту:** Отдает приказ всей колонии переместиться в точку без стрельбы.
- **Режим Перков:** Перехватывает ввод после нажатия кнопки UI для выбора цели способности.
- **Централизация настроек (Inspector):** Все глобальные параметры перков (стоимость, кулдауны, длительность и сила ускорения) выведены в `@export` переменные. Настройка производится один раз в Инспекторе ноды `SelectionManager` в сцене `main.tscn`.

### 6. Система Перков (Perk System)
- **Логика:** Активные способности, требующие **Энергию Перков**.
- **Энергия Перков (⚡):** Пул игрока, пополняемый через захват клеток.
- **Перк "Щит":** Стоимость и КД настраиваются в `SelectionManager`. Дает +50% шанс отражения снарядов.
- **Перк "Ускорение" (Спринт):** Стоимость, КД, длительность и множитель скорости настраиваются в `SelectionManager`. Увеличивает скорость движения всех клеток игрока.
- **UI:** Кнопки с прогресс-баром перезарядки и индикатор общего запаса энергии (⚡).

### 7. Менеджер фракции ИИ (AIFactionManager)
- **Логика:** Центральный мозг каждой компьютерной фракции (Hive Mind).
- **Поведение:** Координирует движение всех клеток фракции как единой колонии.
- **Приоритеты и Скоринг:**
  1. **Тактическая защита (2000px):** Сначала атакует ближайшего врага в радиусе видимости.
  2. **Глобальный захват:** Если врагов рядом нет, выбирает лучшую цель на всей карте по системе весов:
	 - **Враги:** Приоритет слабым (мало энергии) и близким.
	 - **Нейтралы:** Приоритет богатым энергией и близким.
  3. **Патрулирование:** Если целей нет, колония перемещается между случайными точками на карте.
- **Goal Lock:** ИИ фиксирует цель на 4 секунды, чтобы избежать метаний между разными объектами.
- **Energy Threshold:** Если средняя энергия колонии < 55%, ИИ отдает приоритет нейтралам для накопления сил.

### 7. HUD и Интерфейс (HUDLayer)
- **CanvasLayer (100):** Содержит элементы, которые всегда зафиксированы на экране (FPS Counter, PerksUI).
- **FPS Counter:** Обновляется 4 раза в секунду для экономии ресурсов.  

## 🧪 Механики
- **Захват:** Снаряд уменьшает энергию цели. При < 0 клетка меняет владельца.
- **Вклад (Contribution):** Клетка запоминает урон от каждой фракции. При захвате игрок получает свой вклад в общий пул ⚡ энергии.
- **Распад вклада (Decay):** Если клетка не получает урон 10с, вклады уменьшаются на 1 каждые 2с.
- **Лечение:** Прямой клик по своей клетке заставляет других лечить её (без отдачи).
- **Отражение (Reflection):** Шанс снаряда развернуться обратно при попадании в защищенную клетку. Снаряд получает бонус к скорости и меняет цвет/фракцию на отразившую. При захвате клетки все эффекты щита (баффы) сбрасываются.
- **Ускорение (Speed Boost):** При активации перка "Спринт" все клетки игрока получают временный множитель к скорости `move_speed`. Параметры длительности и множителя передаются из `SelectionManager` в момент активации. Эффект визуализируется синим свечением. При захвате клетки эффект сбрасывается.

## ✨ Визуальный "сочный" отклик (Juice)

### 1. Желейная физика (Squash and Stretch)
- **Логика:** Клетки динамически деформируются в зависимости от их скорости. Чем быстрее движение, тем сильнее клетка вытягивается в направлении вектора `velocity`.
- **Сглаживание:** Параметры `visual_stretch` и `visual_angle` плавно интерполируются через `lerp`, исключая дерганье при микро-столкновениях.

### 2. Органическая отрисовка (Procedural Mesh)
- **Мембрана:** Края клетки анимированы синусоидальными волнами (wobble), создавая эффект "живой" капли.
- **Внутренности:** У каждой клетки есть плавающее ядро (Nucleus) с бликом и медленно вращающиеся органеллы.
- **Дыхание:** Все клетки пульсируют в размерах с индивидуальным фазовым сдвигом (breathing animation).

### 3. Реакция на попадание (Hit Feedback)
- **Hit Flash:** При получении урона клетка на мгновение вспыхивает белым (`Color.WHITE`).
- **Impact Wobble:** Сильное хаотичное дрожание мембраны в момент удара, которое быстро затухает.
- **Push:** Снаряд физически отталкивает клетку в сторону полета.

### 4. Снаряды-кометы
- **Голова:** Светящееся ядро с мягким Glow-эффектом.
- **Хвост:** Анимированный виляющий хвост (sin-wave wobble), имитирующий движение микроорганизма.
- **Отражение:** Яркая белая вспышка и инверсия направления при срабатывании щита.

### 5. Клик и Фокус (Targeting Feedback)
- **Contribution Numbers:** Синие цифры (например, +14) рядом с клеткой показывают текущий вклад игрока в её захват.
- **Click Particles:** При клике по карте разлетаются частицы (брызги), цвет которых зависит от типа приказа (зелёный — движение, красный — атака).
- **Target Highlight:** Временная "радарная" подсветка цели (3 секунды) при назначении атаки на врага или нейтрала.
- **Shield Visuals:** Ярко-зеленая мембрана с темными пунктирными линиями, которые быстро бегут по контуру клетки, идеально повторяя её форму.
- **Sprint Visuals:** Мягкое синее свечение (glow) вокруг клетки и усиленная желейная деформация при движении на высоких скоростях.

## 🚀 Оптимизация (Performance)

- **Redraw Throttling:** Функция `queue_redraw()` вызывается не каждый кадр, а через интервал `REDRAW_INTERVAL` (~30 FPS для визуала). Физика при этом работает на полных 60 FPS.
- **Squared Distance:** В модуле `MoverModule` для проверки столкновений сначала используется квадрат расстояния (`length_squared`), что экономит ресурсы на вычислении корней (`sqrt`).
- **Polygon Simplification:** Количество вершин мембраны оптимизировано (20 вместо 32) для снижения нагрузки на отрисовку при больших скоплениях клеток.
- **Static Typing:** Весь код использует строгую типизацию GDScript 2.0 для максимальной скорости исполнения.
